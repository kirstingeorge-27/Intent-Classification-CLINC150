# Intent Classification for Conversational AI

## Overview
This project implements an Intent Classification system using the CLINC150 dataset and BERT transformer model.

## Dataset
CLINC150 (150 intents + 1 out-of-scope class)

## Technologies Used
- PyTorch
- HuggingFace Transformers
- Scikit-learn
- Matplotlib

## Model
Pre-trained BERT (bert-base-uncased) fine-tuned for multi-class classification.

## Results
- Validation Accuracy: ~95%
- F1 Score: ~95%

## How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run training:
   python train.py

3. Evaluate:
   python evaluate.py
